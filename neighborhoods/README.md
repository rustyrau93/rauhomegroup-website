## Page Features

Each page includes:

- **Hero Section** with neighborhood-specific tagline and imagery
- **About Section** with comprehensive neighborhood description
- **Stats Grid** showing median home price, school ratings, and population
- **Lifestyle & Amenities** list with 6+ unique features per neighborhood
- **Who Lives Here** description of community demographics and values
- **CTA Section** encouraging visitors to explore homes
- **Contact Form** pre-populated with neighborhood name
- **SEO Optimization**:
  - Unique meta descriptions
  - Neighborhood-specific keywords
  - Optimized title tags
  - Semantic HTML structure

## Design Consistency

All pages maintain:
- Dark theme (#1A2332) with gold accents (#D4AF37)
- Playfair Display for headings
- Inter font for body text
- Responsive navigation matching main site
- Consistent footer with legal information
- Mobile-responsive layout
